¿Qué es HealthSport?

HealthSport es la aplicación que necesitas para monitorizar los resultados de tu equipo

Se trata de una aplicación web que, mediante registro, permite a los equipos que practican diferentes deportes monitorizar, guardar y obtener estadisticas de las constantes vitales y tiempos obtenidos por sus componentes durante las diferentes competiciones en las que participan

Es posible registrarse como entrenador o deportista. Ambos pertenecerán a un equipo y tendrán acceso a diferentes funcionalidades de acuerdo al rol que asumen.
